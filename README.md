# Pandai Pages

A simple HTML/CSS web project.

## Project Structure

```
pandai-pages-vscode/
├── index.html          # Main HTML file
├── css/
│   └── styles.css      # Stylesheet
├── assets/             # Images and other assets
└── README.md           # This file
```

## Getting Started

1. Open `index.html` in your browser to view the site
2. Edit `css/styles.css` to customize the styling
3. Edit `index.html` to add more content

## Features

- Responsive design
- Modern CSS styling
- Clean HTML structure
